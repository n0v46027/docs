## Our product

We are exited to present you our new product : Octoarea !!\
Octoarea "encapsulate" a service like github for example and check if any changes was proceed on a specific field like an issue on a new repository through api calls \
Once an action is triggered we launch an reaction for example send an email with outlook\
This project was realized with a team of 5 people during an epitech project
